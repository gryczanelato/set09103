    Still not sure how it works?

    Copy

    To copy a URL, open the desired website, then highlight the address in the browser bar, right-click, and select "Copy." Alternatively, you can use the keyboard shortcut Ctrl+C on Windows or Cmd+C on Mac after highlighting the address.

    Paste

    To paste the copied link into a field on the SnapSnapURL page, click in the text field, then right-click in the field and select "Paste." You can also use the shortcut Ctrl+V for Windows or Cmd+V for Mac.
